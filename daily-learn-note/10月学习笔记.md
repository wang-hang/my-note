### vue技巧

调试template 可以挂在一个log方法

```js
// main.js
Vue.prototype.$log = window.console.log;

// 组件内部
<div>{{$log(info)}}</div>
```

### indexOf 和 includes 还可以查找引用类型

```js
const obj = {name: 'www'}
const arr = [obj]

console.log(arr.indexOf(obj)) // 0
console.log(arr.includes(obj)) // true

arr.indexOf({name: 'www'}) // -1 
arr.includes({name: 'www'}) // false

```
